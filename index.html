    Web application VAPT assesment

    The primary target was to Perform a vulnerability assessment and penetration testing (VAPT) on a web application to identify and mitigate potential risks. A thorough method for finding, evaluating, and addressing security flaws in a web application is called VAPT. Penetration testing goes beyond vulnerability assessment by actively attacking flaws to determine their impact and the efficacy of current security solutions. Vulnerability assessment seeks to identify and categorize potential vulnerabilities like Broken access controls, SQL injection vulnarability, and etc...This VAPT assessment uses an automated and manual testing methodology that makes use of industry-standard tools including Burp Suite, Nmap and OWASP ZAP.

    Keylogger using Python

The keylogger was designed as a seamless, background-running application, emphasizing ethical considerations and serving as a tool for understanding malware behavior and anti-detection techniques. This project provided hands-on experience with security tool development, penetration testing, Python programming, and system-level automation, highlighting skills in data encryption and secure authentication.
